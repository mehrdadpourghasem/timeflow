# Screenshots Folder

Add your screenshots here:

1. `screenshot-timer.png` - Screenshot of the Timer view
2. `screenshot-calendar.png` - Screenshot of the Calendar view  
3. `screenshot-analytics.png` - Screenshot of the Analytics view

## How to take screenshots:

1. Run the app locally or visit the live demo
2. Navigate to each view
3. Take a screenshot (use browser DevTools for consistent sizing)
4. Recommended size: 1200x800 pixels
5. Save with the names above

These will appear in your README automatically.
